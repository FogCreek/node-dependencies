## Installation

    sudo npm install -g node-dependencies

## Usage

    cd path/to/project
    node-dependencies

## Options

* `--sort`: Sort with either `alpha` or `urgency`
* `--homepage`: Print the homepage url with each library
* `--pre`: Check packages with non-numeric patch values
